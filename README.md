# Scatch Web App 🛍️

A full-stack e-commerce web application for premium bags built using the MERN stack.  
This project provides a modern shopping experience with authentication, product management, cart functionality, and secure checkout.

## 🚀 Features

- User Authentication (Signup/Login)
- Secure Password Hashing
- JWT-based Authorization
- Product Listing & Details
- Add to Cart Functionality
- Dynamic Cart Management
- Order Management
- Admin Panel for Product Handling
- Responsive UI
- Image Upload Support

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Himanshupal15/CodeAlpha_scatch-web-app.git
```

Go into the project directory:

```bash
cd CodeAlpha_scatch-web-app
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Run the project:

```bash
npm start
```
